## ?? DEPLOYMENT INSTRUCTIONS

### For End Users:

```bash
# 1. Install Copilot CLI
npm install -g @githubnext/github-copilot-cli

# 2. Authenticate (browser required)
github-copilot-cli auth

# 3. Clone repository
git clone https://github.com/alkar1/copilot-vs2026-extension.git

# 4. Build extension
cd copilot-vs2026-extension
dotnet build CopilotExtension/CopilotExtension.csproj -c Release

# 5. Install in VS2026
# Open CopilotExtension.sln in Visual Studio 2026
# Press F5 to build and deploy VSIX
# Or manually install generated .vsix file

# 6. Use in Visual Studio!
# Open C# file, start typing, see suggestions
# Press Tab to accept, Esc to cancel
```
